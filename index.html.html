    ☁️ Configurar sincronización en la nube

    Siga estos pasos una sola vez. Después, los datos se sincronizarán automáticamente entre todos sus dispositivos.

    💡 Antes de empezar: Haga una copia de seguridad JSON de sus datos actuales para no perder nada.
    Paso 1 — Crear cuenta en Firebase
    1. Abra console.firebase.google.com en una pestaña nueva
    2. Inicie sesión con su cuenta de Gmail (o cree una si no tiene)
    3. Acepte los términos si es la primera vez
    Paso 2 — Crear el proyecto
    1. Haga clic en "Crear un proyecto" (o "Agregar proyecto")
    2. Nombre: floreria-leyva
    3. Continuar (puede desactivar Analytics si le pregunta, no es necesario)
    4. Esperar que termine de crear (toma ~30 segundos)
    Paso 3 — Activar Firestore Database
    1. En el menú lateral izquierdo, clic en "Compilación" → "Firestore Database"
    2. Clic en "Crear base de datos"
    3. Elegir ubicación: us-central o southamerica-east1 (Perú)
    4. Modo de seguridad: elegir "Iniciar en modo de prueba" por ahora
    5. Clic en "Habilitar" y esperar unos segundos
    ⚠️ Nota de seguridad: "Modo prueba" expira en 30 días. Después hay que ajustar reglas (se lo explico cuando llegue el momento).
    Paso 4 — Crear app web y obtener credenciales
    1. En la página principal del proyecto, busque el ícono </> (o "Agregar app" → Web)
    2. Alias: floreria-leyva-web
    3. NO marque "Firebase Hosting" (no es necesario)
    4. Clic en "Registrar app"
    5. Verá un bloque de código con firebaseConfig = { ... }
    6. Copie esos 6 valores (apiKey, authDomain, projectId, storageBucket, messagingSenderId, appId)

    Registro de inversiones

    — Injerto · techado de láminas · infraestructura
    💎
    ¿Por qué las inversiones son distintas a los gastos?
    Las inversiones grandes como el injerto o el techado de láminas no son gastos corrientes. Duran varios años y su costo debe distribuirse (amortizarse) entre todo ese tiempo. El sistema calcula automáticamente cuánto representa esa inversión por año, mes y semana, para que el costo real por rosa refleje la realidad del cultivo sin distorsiones.

          ☁️ Sincronización en la nube (Firebase)

          Con Firebase (gratis para uso normal) los datos se sincronizan entre el celular, la computadora y cualquier dispositivo en tiempo real. Si su papá registra una venta en el celular, aparece al instante en su computadora.

          💡 Mientras no configure Firebase, el sistema funciona 100% localmente. Puede usar la función de respaldo JSON para mover datos entre dispositivos manualmente.

          🧹 Limpieza de datos por fundo

          Borre solo los datos de un fundo específico. Los otros fundos quedan intactos.

          Borrar TODOS los datos de TODOS los fundos. Esta opción elimina absolutamente todo y requiere escribir una palabra clave para confirmar.
